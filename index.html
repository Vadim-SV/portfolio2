latest work

Crown Shop

The web app is designed to be scalable as ASOS, which is achieved by implementing REDUX state management. The simless LogIn and Registration is done utilising Firebase API to allow  users sign in or signup quickly. Users also can add items to the shopping cart and come back later  to checkout as the user session is being saved. Shopping items as well as users are hosted in the Firebase database which also provides Google SignIn feature. This is a single page app which is achieved by using React-router to simulate the existence of multiple pages.Checkout can be done quickly and securely using STRIP API.

Tech stack used:
  • ReactJS;
  • React-router;
  • ReactSASS;
  • RESTful API;
  • Stripe API;
  • Styled components;
  • Firebase cloud databases for users and shop items;
  • Google Sign In/Signup feature;

Londoners

React JS based app that utilises third party API to fetch and return avatars for cards with names and addresses.
It is also has Search function allowing users to type something in search field triggering searching function that sorts cards by letters typed by users.
The app utilises Redux to manage state, particularly Search function. Actions created to listen to an event of users typing something in search filed triggering reducers which then trigger state change.
